# svelte-template

my personal svelte starter template. clone it, nuke the git repo (create ur own), and go.

```bash
pnpm install
pnpm dev
```

### stuff is in here

- **svelte 5** — no boilerplate just a counter
- **tailwind css v4** — `@theme`, `@utility`, no extra config
- **zero config hell** — like 2 config files and 3 dependencies

no sveltekit, no bloat. just a blank page with a button and a dark theme.

edit `src/App.svelte` and start building.
